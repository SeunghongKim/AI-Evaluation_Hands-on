비생성형 AI (ML/DL) 평가·검증

전통적인 머신러닝 모델의 데이터 편향과 모델 공정성을 평가합니다 · UCI Adult Income Dataset 활용

Notebook 01 · ML · 데이터 공정성
📊 데이터 표본 편향 평가
학습데이터·검증데이터·시험데이터가 원천데이터(모집단)를 통계적으로 잘 대표하는지 PSI 지표로 측정합니다. 성별·학력·인종 보호변수를 기준으로 분포 편향을 정량화합니다.
데이터 공정성 ML PSI
평가 지표: PSI (Population Stability Index)
통과 기준: PSI < 0.25
데이터셋: UCI Adult Income (48,842개)

생성형 AI (LLM) 평가·검증

대규모 언어 모델의 데이터 대표성, 텍스트 유창성, 유해성을 평가합니다 · IMDb · DistilGPT-2 활용

Notebook 04 · LLM · 데이터 공정성
🌐 데이터 대표성 편향 평가
LLM 학습 데이터의 성별 대명사·직업 언급 분포가 특정 속성에 편향되지 않았는지 Shannon Entropy로 측정합니다. IMDb 영화 리뷰 5,000개를 분석합니다.
데이터 공정성 LLM Entropy
평가 지표: Shannon Entropy (정규화 0~1)
통과 기준: 정규화 엔트로피 > 0.7
데이터셋: IMDb Review (HuggingFace)

핵심 평가 지표 요약

6개 노트북에서 사용되는 평가 지표와 통과 기준을 한눈에 확인하세요

평가 지표별 통과 기준

노트북 평가 지표 공식/설명 통과 기준 대상
#1 표본 편향 PSI Σ (P_i − A_i) × ln(P_i / A_i) PSI < 0.25 성별·학력·인종
#2 라벨링 편향 4/5 Rule min(rate) / max(rate) 비율 ≥ 0.80 성별·교육수준
#3 모델 공정성 FPR / TPR / FDR Parity |그룹 A − 그룹 B| Gap < 0.10 성별
#4 대표성 편향 Shannon Entropy H = −Σ p(x) log₂ p(x) 정규화 엔트로피 > 0.7 성별·직업·길이
#5 유창성 Fluency Score + PPL 규칙 기반 + exp(Cross-Entropy Loss) Score ≥ 3.5, PPL < 100 생성 텍스트
#6 유해성 Toxicity Score Detoxify 6개 차원 최대값 Score < 0.20 생성 텍스트

PSI 해석 기준

우수
PSI < 0.10
변화 없음 · 분포 안정
통과
0.10 ~ 0.25
소폭 변화 · 주의 관찰
실패
PSI ≥ 0.25
큰 변화 · 편향 존재

시작하기

로컬 환경에서 노트북을 직접 실행해보세요

4단계로 바로 시작

1

저장소 클론

GitHub에서 프로젝트를 로컬로 복제합니다.

2

가상환경 생성

Python 가상환경을 만들고 의존성을 설치합니다.

3

Jupyter 실행

jupyter notebook 명령으로 실습 환경을 엽니다.

4

순서대로 실행

1번 노트북부터 셀을 순서대로 실행합니다.

# 1. 저장소 클론
git clone https://github.com/your-username/ai-evaluation-lab.git
cd ai-evaluation-lab

# 2. 가상환경 생성 및 활성화
python -m venv venv
source venv/bin/activate        # macOS/Linux
# venv\Scripts\activate         # Windows

# 3. 의존성 설치
pip install -r requirements.txt

# 4. Jupyter 실행
jupyter notebook notebooks/
