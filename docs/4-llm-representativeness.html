1. Shannon Entropy란?

정보 이론에서 엔트로피(Entropy)는 확률 분포의 불확실성·다양성을 측정하는 지표입니다. 데이터 대표성 평가에서 높은 엔트로피는 균형 잡힌 분포를, 낮은 엔트로피는 특정 그룹의 과대표현을 의미합니다.

H(X) = −Σ p(x) × log₂ p(x)
p(x): 각 카테고리의 출현 확률  ·  H(X): 엔트로피 (bits)
정규화 엔트로피 = H(X) / log₂(n)  (0~1 범위, 1 = 완전 균등)

Excellent

매우 균등하고 다양한 분포 → 편향 없음

정규화 엔트로피 > 0.9

Good

적절한 다양성 → 경미한 편향

0.7 ~ 0.9

Fair

불균등한 분포 → 중간 수준 편향

0.5 ~ 0.7

Poor

심각한 불균등 → 강한 편향 존재

< 0.5

3. 구현 코드

3.1 성별 대명사 추출

def extract_gender_pronouns(text):
    """정규표현식으로 성별 대명사 카운트"""
    text_lower = text.lower()

    male_pronouns   = ['he', 'him', 'his', 'himself']
    female_pronouns = ['she', 'her', 'hers', 'herself']

    male_count   = sum(len(re.findall(r'\b'+p+r'\b', text_lower)) for p in male_pronouns)
    female_count = sum(len(re.findall(r'\b'+p+r'\b', text_lower)) for p in female_pronouns)

    return {'male': male_count, 'female': female_count}

3.2 엔트로피 계산

def calculate_entropy(counts_dict):
    """정규화 엔트로피 계산"""
    total = sum(counts_dict.values())
    probs = np.array([c / total for c in counts_dict.values()])
    probs = probs[probs > 0]  # log(0) 방지

    entropy = -np.sum(probs * np.log2(probs))
    max_entropy = np.log2(len(counts_dict))
    norm_entropy = entropy / max_entropy if max_entropy > 0 else 0

    return entropy, max_entropy, norm_entropy

성별 대명사 편향 해석

영화 리뷰 데이터에서 남성 대명사(he/him/his)가 여성 대명사(she/her/hers)보다 약 2배 많이 등장합니다. 이는 헐리우드 영화가 역사적으로 남성 주인공 중심이었음을 반영합니다. 이러한 데이터로 LLM을 학습하면 "주인공 = 남성" 고정관념이 강화될 수 있습니다.

6. 개선 방향

데이터 수집 단계

  • 다양한 장르·연대·국가의 영화 리뷰 포함 (여성 주인공 영화 추가)
  • 의도적으로 성별 균형 잡힌 리뷰 수집
  • 단일 도메인(영화) 탈피 → 다양한 텍스트 소스 추가

데이터 증강

  • 성별 교체(Counterfactual Augmentation): "He is a doctor" → "She is a doctor" 변형 텍스트 추가
  • 합성 데이터 생성: LLM으로 소수 그룹 중심 텍스트 생성
# 성별 교체 증강
def gender_swap_augment(text):
    replacements = {
        ' he ': ' she ', ' him ': ' her ',
        ' his ': ' her ', ' himself ': ' herself '
    }
    for male, female in replacements.items():
        text = text.replace(male, female)
    return text
