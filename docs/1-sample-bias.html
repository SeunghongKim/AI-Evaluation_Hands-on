3. PSI (Population Stability Index)

PSI = Σ (Pi − Ai) × ln(Pi / Ai)
Pi: 원천데이터의 i번째 그룹 비율  ·  Ai: 학습데이터의 i번째 그룹 비율  ·  n: 총 그룹 수
PSI 범위해석판정
PSI < 0.10분포 변화 없음✅ 우수
0.10 ≤ PSI < 0.25소폭 변화⚠️ 통과 (주의)
PSI ≥ 0.25큰 변화 → 편향 존재❌ 실패

구현 코드

def calculate_psi(population_dist, sample_dist):
    """PSI 계산 함수"""
    pop_prop = population_dist / population_dist.sum()
    sample_prop = sample_dist / sample_dist.sum()

    all_groups = set(pop_prop.index) | set(sample_prop.index)
    psi = 0

    for group in all_groups:
        pop_p = pop_prop.get(group, 0.0001)   # 0 방지
        smp_p = sample_prop.get(group, 0.0001)
        psi += (pop_p - smp_p) * np.log(pop_p / smp_p)

    return psi

6. 편향 시 개선 방향

PSI ≥ 0.25 발생 시 조치

  • 재샘플링: Stratified Sampling으로 보호변수 비율 유지
  • 가중치 조정: 과소/과대 표현된 그룹에 샘플 가중치 부여
  • 데이터 수집: 부족한 그룹의 데이터를 추가로 수집
  • 오버샘플링: SMOTE 등으로 소수 그룹 합성 데이터 생성
