⚖️ 데이터 라벨링 편향 평가

4/5 Rule (80% 규칙)로 보호변수 그룹별 긍정 라벨 비율이 공정한지 검증합니다.

1. 개요: 4/5 Rule이란?

4/5 Rule은 미국 EEOC(Equal Employment Opportunity Commission)가 고용 차별 판단에 사용하는 통계적 기준입니다. AI 모델에서는 보호변수 그룹 간 긍정 라벨 비율의 격차를 측정하는 데 활용됩니다.

4/5 Rule Ratio = min(그룹별 긍정 라벨 비율)
4/5 Rule Ratio = max(그룹별 긍정 라벨 비율)
비율 ≥ 0.80 (80%) → PASS (공정)  |  비율 < 0.80 → FAIL (편향 존재)

왜 80%인가?

최소 그룹의 선발률이 최고 그룹의 80% 이상이면 "통계적으로 의미 있는 차별"로 보지 않는다는 경험적 기준입니다. 금융·채용·의료 AI에서 공정성 검증의 기준선으로 널리 사용됩니다.

3.2 4/5 Rule 계산 함수

def calculate_labeling_bias(df, protected_attr, label_col='income_binary'):
    """4/5 Rule로 라벨링 편향 계산"""
    positive_rates = df.groupby(protected_attr)[label_col].mean()

    min_rate = positive_rates.min()
    max_rate = positive_rates.max()
    four_fifths_ratio = min_rate / max_rate if max_rate > 0 else 0

    return {
        'positive_rates': positive_rates,
        'four_fifths_ratio': four_fifths_ratio,
        'is_fair': four_fifths_ratio >= 0.8
    }

5. 결과 해석

⚠️ 중요한 해석: 편향 ≠ 모델 오류

  • UCI Adult Income은 1994년 미국 사회의 실제 소득 분포를 반영합니다.
  • 성별 편향은 당시 남녀 임금 격차라는 역사적·사회적 불평등이 데이터에 그대로 담긴 것입니다.
  • 이러한 데이터로 모델을 학습하면 과거의 차별이 미래에 재생산될 수 있습니다.
  • 4/5 Rule은 "이 데이터로 학습해도 되는가?"를 판단하는 첫 번째 관문입니다.

6. 개선 방향

데이터 수집 단계

  • 보호변수별 균형 잡힌 샘플링 적용
  • 명확한 라벨링 가이드라인으로 라벨러 주관 편향 최소화
  • 다양한 배경의 라벨러 활용

데이터 전처리 단계

  • 리샘플링: SMOTE, Oversampling으로 과소표현 그룹 보완

  • 공정성 라이브러리: Fairlearn, AIF360 활용
